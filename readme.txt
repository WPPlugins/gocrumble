== Description ==
 
Crubmle is a free simple white-label file sharing platform that provides an easy way to collaborate and share files between employees and your contacts.

This plugin provides a widget that allows your Crumble contacts to enter the e-mail address and password to access their shared files directly from your wordpress site. 

All you have to do is [__setup a Crumble account__](https://clients.gocrumble.com/signup)

__Easy Sharing__

- Share content through a unique token
- Support for expiration and password protection for shares
- Separate contact login to access all contact shares

__White-labeling__ 

- Use your own logo, icon and color theme 
- Upload your cover photo
- Own domain
- Responsive design
- Use your e-mail servers

__Secure__

- Server Side encryption for your files
- Communication through secure link
- Member role based access

 
== Installation ==
 
This section describes how to install the plugin and get it working.

1. Setup a [Crumble account](https://clients.gocrumble.com/signup)
2. Upload few files and create contacts with password option enabled
3. Install the Wordpress Crumble plugin
4. Go to Appearance -> Widgets and add your widget to your site
5. Specify the API Endpoint as the URL to your Crumble account e.g. https://[your-company-domain].gocrumble.com
